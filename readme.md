<h1 align="center">🎙️ Automatic Announcement Quality Analyzer (AAQA)</h1> <p align="center"> <b>An AI + DSP-based system to evaluate the quality of public announcements in real-time.</b> </p> <hr> <h2>👨‍💻 Team Members</h2> <ul> <li><b>Abhinav Kesarwani</b> (23AI002)</li> <li><b>Ajitesh Channa</b> (23AI006)</li> <li><b>Anant Jain</b> (23AI007)</li> <li><b>Rishav Kumar</b> (23AI043)</li> </ul> <hr> <h2>📌 Project Overview</h2> <p> The <b>Automatic Announcement Quality Analyzer (AAQA)</b> is a system designed to evaluate the quality of public announcement audio using <b>Digital Signal Processing (DSP)</b> techniques. It analyzes uploaded or recorded audio and provides a quality score, feedback, and transcription. </p> <p> The system is particularly useful for <b>transportation systems</b> such as railway stations and airports, where clear communication is critical. </p> <hr> <h2>🚀 Features</h2> <ul> <li>🎧 Audio Upload & Recording</li> <li>📊 Real-time Quality Scoring (0–100)</li> <li>📈 DSP Metrics Visualization</li> <li>🧠 Non-intrusive Audio Analysis (No reference signal needed)</li> <li>📝 Automatic Speech Transcription</li> <li>📉 Noise & Clarity Feedback</li> </ul> <hr> <h2>⚙️ Tech Stack</h2> <h3>Frontend</h3> <ul> <li>Next.js</li> <li>React</li> <li>WaveSurfer.js</li> </ul> <h3>Backend</h3> <ul> <li>FastAPI</li> <li>Python</li> </ul> <h3>DSP & Audio Processing</h3> <ul> <li>NumPy</li> <li>SciPy</li> <li>SpeechRecognition (Google API)</li> </ul> <h3>Experimental (Not Deployed)</h3> <ul> <li>Wav2Vec2 (Transformers)</li> <li>PyTorch</li> </ul> <hr> <h2>🧠 DSP Concepts Used</h2> <ul> <li><b>Signal-to-Noise Ratio (SNR)</b></li> <li><b>Spectral Flatness</b></li> <li><b>Zero Crossing Rate (ZCR)</b></li> <li><b>Formant Energy Ratio</b></li> </ul> <hr> <h2>📐 Scoring Methodology</h2> <p> The system uses a <b>non-linear scoring model</b>: </p> <ul> <li>📌 Sigmoid scaling for SNR</li> <li>📌 Power scaling for formant energy</li> <li>📌 Log scaling for spectral flatness</li> </ul> <p> Final score is computed as a weighted combination of these features. </p> <hr> <h2>🏗️ System Architecture</h2> <pre> Audio Input → Preprocessing → DSP Feature Extraction → Scoring Engine → Output (Score + Feedback + Transcription) </pre> <hr> <h2>📊 Output</h2> <ul> <li>🎯 Quality Score (0–100)</li> <li>📌 Label (Good / Moderate / Poor)</li> <li>🔊 Noise Level</li> <li>🧾 Transcription</li> <li>📊 DSP Metrics</li> </ul> <hr> <h2>⚠️ Limitations</h2> <ul> <li>No adaptive learning (heuristic model)</li> <li>Requires tuning for different environments</li> <li>ML model not fully integrated</li> </ul> <hr> <h2>🔮 Future Scope</h2> <ul> <li>Integration of Deep Learning (Wav2Vec2)</li> <li>Dataset-based training</li> <li>Hybrid DSP + AI model</li> <li>Real-time deployment in railway systems</li> </ul> <hr> <h2>🛠️ Setup Instructions</h2> <h3>Backend</h3> <pre> cd backend pip install -r requirements.txt uvicorn main:app --reload </pre> <h3>Frontend</h3> <pre> cd frontend npm install npm run dev
